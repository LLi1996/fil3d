2D filament mask containers (MaskNodes)¶

To use, do:

from fil3d import MaskObjNode

node = MaskObjNode(mask, corners, v_channel)
class fil3d.structs.MaskObjNode(mask_obj, corners, v_slice_index)¶

The MaskObjNode class is made to contain a single mask and its corners.

The mask and corner can be produced by FilFinder or any other filament finding code. You can also manually input masks and corners .

Each instance takes in:

  1. a mask (2d np bit array)

  2. the corners of the mask indicating where the mask is located

  3. the index of the velocity channel where the mask is located

It is important to note the difference between the area of a mask and the masked area: since the masks are always rectangles, the area of a mask is just calculated from dimensions of the corners. The masked area, however, is calculated by counting the amount of pixels that’s masked (marked TRUE in the np array).

check_mask_overlap(other_node, overlap_thresh)¶

Check if there are overlaps between other_node and self.

First check if the corners of self.mask and other_node.mask overlap. If the corners do overlap (meaning some part of the two squares overlap), we then check if actual masks overlap. To check for actual overlap, a combined AND mask is first made, and the masked area calculated. The masked area of the combined AND mask is then compared to the masked area of the input masks.

Parameters

  • other_node (MaskObjNode) – Node to check for overlap with self.

  • overlap_thresh (float) – Overlap fraction expressed as decimal.

Returns

True if the the overlap between the combined AND mask and _either_ of the masks is greater than overlap_thresh.

combine_mask(other_node, merge_type='AND')¶

Combine self.mask with other_node.mask.

The corners of the 2 masks are first matched to find the smallest square that contains both masks, a new mask is then made to that dimension. Elements of the new mask are filled basked on the merge_type (‘AND’ or ‘OR’).

Parameters

  • other_node (MaskObjNode) – Node whos mask we will combine with.

  • merge_type (str) – Type of merge. Choices: ‘AND’, ‘OR’. Defaults to ‘AND’.

Returns

Combined mask.

Return type

np.array
